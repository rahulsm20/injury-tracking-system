### Tech stack 
- Next.js
- TailwindCSS
- Ant design
- Auth0
- Prisma
- GraphQL (Apollo Server)
- PostgreSQL

### Steps to run locally 
1. Clone this repo 
    ```
    git clone https://github.com/rahulsm20/injury-tracking-system
    ```
2. Enter folder
    ```
    cd injury-tracking-system
    ```
3. Install packages
    ```
    npm install 
    ```
4. Add the following environment variables
    ```
    DATABASE_URL 
    AUTH0_SECRET 
    AUTH0_BASE_URL
    AUTH0_ISSUER_BASE_URL
    AUTH0_CLIENT_ID
    AUTH0_CLIENT_SECRET
    ```
5. Run project
   ```
   npm run dev
   ```
